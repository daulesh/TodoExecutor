## Getting Started



### Creating your account

1. Visit `http://localhost:3000/welcome`
2. Click **Get Started Free** → **Create Account**
3. Register with your email, username, and password



### First login

After logging in, you land on the **Dashboard** — your daily command center showing:

- Today's date and week view
- Your AI Daily Briefing (personalized morning greeting)
- All tasks scheduled for today
- Category filter chips

---



## Dashboard

The dashboard shows **today's tasks** by default.

### Week Calendar

- Click any day in the week strip to jump to that day's task view
- Use **‹** / **›** arrows to navigate weeks
- Today is always highlighted in purple



### Category Chips

Filter today's tasks by category using the chip bar:

- **All Tasks** — shows everything
- **Health / Work / Personal / etc.** — filters to that category



### Completing a Task

- Click the **circle icon** left of any task to mark it complete
- A dialog appears asking for optional completion notes and actual time spent
- This data feeds your **Analytics Coach**



### Clicking a Task

- Click the task title/description area to open the **Task Edit page**
- From there you can edit, reschedule (with mandatory audit reason), delete, or generate AI subtasks

## Tasks & Categories



### Viewing All Tasks by Category

Navigate to **Tasks (Categories)** in the sidebar to see all your tasks grouped by category as accordion sections.

### Creating a Category

1. Go to the **Analytics** page or use the AI Planner to auto-create one
2. Or navigate directly to create a category from the Tasks page



### Creating a Task Manually

1. Click the **+ FAB button** (bottom-right, purple circle with + icon) on the Dashboard or Date view
2. Fill in:
  - **Title** (required)
  - **Description** (optional — supports markdown checklist items)
  - **Target Date** (required)
  - **Category** (optional)
  - **Start/End Time** and **Estimated Duration** (optional, for scheduling)



### Rescheduling a Task (Audit Trail)

1. Open a task from the edit page
2. Change the **Target Date** field
3. Click **Save Changes** — a dialog forces you to provide a **reason** for the change
4. The reason is logged in the **Rescheduling Audit History** section of that task

> **Why is this important?** The Rescheduler Agent reads this history to identify chronically delayed tasks and optimize your schedule intelligently.

## AI Daily Briefing

Every time you open the Dashboard, an **AI-powered greeting card** appears at the top, generated by Gemini based on:

- How many tasks you have scheduled for today
- How many tasks are overdue
- How many tasks you completed yesterday
- A few upcoming task titles

**Example:**

> *"Good morning! You have 4 tasks lined up today including your Kaggle EDA script — tackle that first to maintain your 3-day streak!"*

This briefing is **non-blocking** — the page loads instantly and the briefing appears when ready.

## AI Goal & Task Planner

> **Best for:** Starting a new project, preparing for an event, or planning any multi-step goal.



### How to use it

1. Go to the **Tasks (Categories)** page
2. Scroll to the **"AI Goal & Task Planner"** card
3. Type your goal in natural language:
  - *"Plan my preparation for a machine learning Kaggle competition over the next 3 weeks"*
  - *"Help me organize my gym routine and meal prep for next month"*
  - *"I need to launch a side project MVP in 2 weeks"*
4. Click **Generate Plan**
5. Wait 10–30 seconds (the agent is working)
6. The agent will:
  - Check your existing categories
  - Create a new category if needed
  - Break your goal into 4–7 tasks with specific target dates
  - Return a written summary of the plan

> **Tip:** After the plan is created, refresh the Tasks page to see the new category and tasks populate!

## AI Analytics Coach

> **Best for:** Understanding your productivity patterns and getting actionable coaching advice.



### How to use it

1. Go to the **Analytics & Export** page
2. Scroll to the **"AI Coach"** card
3. Type a question or leave the default prompt:
  - *"Analyze my productivity and give me coaching advice"*
  - *"Which category am I neglecting the most?"*
  - *"What is my completion rate this week?"*
4. Click **Get AI Insights**
5. The agent queries your actual database (task counts, completion rates, reschedule history, overdue counts by category) and returns professional coaching text

**Sample output:**

> *"Your overall completion rate is 68%, which is strong! However, your 'Health' category has only 2 of 9 tasks completed — this is your biggest opportunity. You've rescheduled 5 tasks in the past week, mostly in the 'Work' category, suggesting possible overcommitment. Consider reducing tasks per day in Work from 4 to 2-3."*

## AI Chat Assistant

> **Best for:** Quick natural language interaction without navigating to specific pages. The most powerful interface.



### How to open it

Click the **✨ purple AI button** floating in the **bottom right corner** of any page (second button from bottom).

### What you can say


| Intent           | Example Message                                  | Agent Invoked         |
| ---------------- | ------------------------------------------------ | --------------------- |
| **Analytics**    | *"How am I doing this week?"*                    | Analytics Coach       |
| **Analytics**    | *"What's my completion rate?"*                   | Analytics Coach       |
| **Planning**     | *"Plan my exam prep for next month"*             | Task Planner          |
| **Planning**     | *"Create a workout schedule for me"*             | Task Planner          |
| **Rescheduling** | *"I'm overwhelmed, reschedule my overdue tasks"* | Rescheduler           |
| **Rescheduling** | *"Optimize my schedule"*                         | Rescheduler           |
| **General**      | *"Hello! What can you help me with?"*            | Orchestrator (direct) |




### How it works

The **Orchestrator Agent** reads your message and decides:

- Should this go to **Analytics**? → delegates to Analytics Coach
- Should this go to **Planning**? → delegates to Task Planner
- Should this **reschedule tasks**? → delegates to Rescheduler
- Is this just conversation? → responds directly

No routing configuration needed — it's all automatic.

## AI Subtask Generator

> **Best for:** When you have a task but don't know how to break it down.



### How to use it

1. Click any task from the Dashboard or Tasks page to open the **Task Edit page**
2. Scroll down to the **"AI Subtask Generator"** card (below the edit form)
3. Click **Generate Actionable Subtasks**
4. The AI analyzes your task title and description and returns 3–5 concrete steps
5. Click **+ Add to Desc** next to any subtask to append it to your task description as:
  ```
   - [ ] Research preprocessing techniques
   - [ ] Write feature engineering script
  ```
6. Save the task to persist the updated description

## Power User Tips



### 1. Build the Audit Habit

Every time you reschedule a task, write a *specific* reason:

- ✅ *"Client feedback delayed the requirement"*
- ❌ *"Pushed back"*

The Rescheduler Agent reads th suggestions.

### 2. Use Stese logs and uses them to make smarter dateructured Descriptions

Write task descriptions with action items:

```
- [ ] Review the dataset columns
- [ ] Handle null values in Age column
- [ ] Normalize numerical features
```

The AI Subtask Generator can automatically generate these for you!

### 3. Chat First, Navigate Second

Instead of browsing to find the Analytics page or Tasks page, just open the AI Chat and type what you need. The Orchestrator routes your request automatically.

### 4. Run the Rescheduler Periodically

Every few days, open the Chat Assistant and say: *"Reschedule my overdue tasks"*. The agent will redistribute them intelligently, avoiding overloading any single day.

### 5. Review the Daily Briefing

The morning greeting contains real data. If it says you have 3 overdue tasks, act on it before adding new ones.
